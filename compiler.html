Your Momma Compiler

Compiler Project - Your Momma Compiler

 											

This program is a multi-pass compiler for the following simple grammar:

For Instructions: READ-ME	

==================================================================
==								==
==			GRAMMAR					==
==								==
==================================================================


Program         ::== Statement $

Statement       ::== print ( Expr )							 
	        ::== Id = Expr
                ::== VarDecl
                ::== { StatementList }
                ::== ifExpr
                ::== whileExpr
                ::== comment

StatementList   ::== Statement StatementList
	 	::== ε

Expr 		::== IntExpr
		::== StringExpr
		::== Id

ifExpr          ::== if( condExpr ){ StatementList }

whileExpr       ::== while( condExpr ){ StatementList }

condExpr        ::== id condOp id
		::== id
-----------------------------------------------------
NOTE: ID FOR CONDOP CAN ONLY BE USED FOR INTEGERS   |					
-----------------------------------------------------
comment  	::== // comExpr
		::== /* comExpr */
comExpr 	::== char comExpr
		::== digit comExpr
		::== op comExpr
		::== condOp comExpr
		::== space comExpr
		::== ε

IntExpr 	::== digit op Expr									   
		::== digit

StringExpr      ::== " CharList "

CharList 	::== Char CharList
		::== Space CharList				
		::== ε

VarDecl 	::== Type Id

Type 		::== int | string
Id 		::== Char									 

Char 		::== a | b | c ... z
Space		::== the space character
digit	 	::== 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | 9 | 0
op 		::== + | -
condOp 		::== ==
